# 🎓 Agrigator — агрегатор курсов и репетиторов

Демо-сайт агрегатора подготовки к международным экзаменам (IELTS, TOEFL, SAT, ACT, IB, A-Level, AP, GMAT, GRE, ЕНТ, NUET) с полноценным бэкендом: Node.js + SQLite (встроенный `node:sqlite`, без npm-зависимостей). Все данные — пользователи, курсы, репетиторы, отзывы, записи, лиды парсера — хранятся в SQL-базе `agrigator.db`.

## Запуск

```bash
cd ~/Documents/Agrigator
node server.js          # или: npm start
# открыть http://localhost:8765
```

Нужен Node.js ≥ 22.5. При первом запуске база создаётся и заполняется демо-данными автоматически.

## Что внутри

**Каталог** — 16 курсов и 10 репетиторов, фильтры по экзамену, типу, формату (онлайн/оффлайн/гибрид), городу, цене, рейтингу, пробному уроку и AI-верификации; поиск, сортировки, «показать ещё», избранное ❤️ и сравнение ⚖️ до 3 позиций.

**Подписка Pro** (демо-оплата картой) — закрывает «критические данные»: реальную статистику результатов «до/после» с выборками, прямые контакты (Instagram/Telegram/WhatsApp/телефон) и AI-отчёты со звонков.

**Аккаунты** — регистрация/вход, профиль с фото-аватаром, записями на уроки, AI-отчётами и верифицированными экзаменами. Админ может править всё.

**Профили курсов/репетиторов** — фото, видео-презентация, контакты, бесплатный/платный пробный урок, планирование звонка по слотам (ссылка Calendly), отзывы с оценками и баллами «до → после», бейджи «проверенный студент».

**AI-фичи**
- 🤖 **AI-ассистент звонков** (`#/ai-call`) — главная фича: ИИ «сидит» на демо-уроке, в реальном времени пишет транскрипт, отмечает ошибки/успехи студента, двигает метрики (грамматика, словарный запас, беглость) и после звонка генерирует отчёт с рекомендациями и прогнозом балла.
- 🛡️ **AI-верификация** (`#/verify`) — подтверждение IELTS/SAT score reports (OCR → сверка с базой → анти-фрод → бейдж «✓ verified»).
- 🎯 **AI-подбор** (`#/match`) — квиз из 5 вопросов, ранжирование каталога с match-процентом и причинами.

**Админ-панель** (`#/admin`) — дашборд с метриками и MRR, CRUD курсов/репетиторов, тумблеры «Реклама» и «AI-verified», модерация отзывов, управление пользователями (роли, планы), **парсер объявлений** из Instagram/OLX/Telegram (вставь текст объявления — ИИ извлечёт экзамен, цену и контакты и создаст лид → карточку каталога; кнопка «Автопоиск» имитирует краулер), экспорт/сброс базы.

**Реклама** — спонсорские карточки с лентой «Реклама» поднимаются в выдаче и показываются блоком на главной.

**Дизайн** — светлая/тёмная тема (переключатель 🌙 в шапке), адаптив под мобильные.

## Бэкенд и SQL

`server.js` — HTTP-сервер без зависимостей: отдаёт статику и REST API под `/api`. База — SQLite, таблицы: `users` (пароли захэшированы scrypt), `sessions` (Bearer-токены), `items` (курсы и репетиторы), `reviews`, `bookings`, `leads`, `reports`, `verified_exams`.

Основные эндпоинты:

| Метод и путь | Что делает | Доступ |
|---|---|---|
| `GET /api/bootstrap` | все данные для фронта одним запросом | все |
| `POST /api/auth/register · login · logout` | аккаунты и сессии | все |
| `POST /api/items`, `DELETE /api/items/:type/:id` | создание/правка/удаление курсов и репетиторов | админ |
| `POST /api/reviews`, `DELETE /api/reviews/:id` | отзывы (+ автопересчёт рейтинга) | юзер / админ |
| `POST /api/favorites/toggle` | избранное | юзер |
| `POST /api/bookings` | запись на урок/звонок | все |
| `POST /api/leads`, `PATCH /api/leads/:id` | лиды парсера объявлений | админ |
| `PATCH /api/users/:id`, `DELETE /api/users/:id` | профиль, планы, роли | сам / админ |
| `POST /api/me/reports · verified` | AI-отчёты и верификация | юзер |
| `POST /api/admin/reset`, `GET /api/admin/export` | сброс к демо-данным, экспорт JSON | админ |

Посмотреть базу напрямую: `sqlite3 agrigator.db "SELECT id, type, json_extract(data,'$.title') FROM items;"`

## Структура

```
server.js           — бэкенд: Node http + SQLite, REST API, сидинг
agrigator.db        — SQL-база (создаётся при первом запуске)
index.html          — каркас страницы
css/styles.css      — стили + темы
js/data.js          — демо-данные для сидинга базы
js/store.js         — клиент API: кэш + fetch-запросы
js/ui.js            — общие компоненты (карточки, модалки, тосты, premium-lock)
js/views-public.js  — главная, каталог, детальные, сравнение
js/views-user.js    — вход/регистрация, профиль, тарифы
js/views-ai.js      — AI-подбор, AI-ассистент звонков, AI-верификация
js/views-admin.js   — админ-панель и парсер объявлений
js/app.js           — роутер и шапка
```

Сбросить все данные к исходным: Админ-панель → Настройки → «Сбросить к демо-данным» (или удалить файл `agrigator.db` и перезапустить сервер).
